# 🚀 Swag Labs UI Test Automation Framework

> A complete UI Test Automation Framework for the **Swag Labs** web application built using **Java, Selenium WebDriver, TestNG, Maven, Page Object Model (POM), PageFactory, JSON Test Data, and Allure Reports**.

---

## 📌 About The Project

This project demonstrates a complete UI Test Automation Framework developed for the **Swag Labs** web application.

The framework follows industry-standard automation testing practices with a clean and maintainable architecture. It automates the complete purchasing workflow from user login to successful checkout while ensuring code reusability, scalability, and readability.

---

- ✅ Page Object Model (POM)

# ▶️ How To Run

## Clone Repository

```bash
git clone https://github.com/muhammed-elgarf/Swag-Labs-UI-Automation.git
```

## Navigate To Project

```bash
cd Swag-Labs-UI-Automation
```

## Install Dependencies

```bash
mvn clean install
```

## Execute Test Suite

```bash
mvn test
```

## Generate Allure Report

```bash
allure serve allure-results
```
